## Usage:

1. Go to **Blogs** page
2. Click **Add Blog**
3. Use the rich text editor toolbar:
   - **Bold/Italic/Underline** buttons
   - **List** buttons for bullets/numbers
   - **Alignment** buttons
   - **Format** dropdown for headings
   - **Link** button for URLs
   - **Code** button for HTML view
4. Save and publish!

- **Ctrl+I**: Italic
- **Ctrl+U**: Underline
- **Ctrl+Z**: Undo
- **Ctrl+Y**: Redo
